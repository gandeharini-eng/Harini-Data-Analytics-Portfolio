# COVID-19 Data Exploration (SQL Project)

This project explores global COVID-19 data using SQL.

Skills Demonstrated:
- Joins
- Window Functions
- CTE (Common Table Expressions)
- Temporary Tables
- Aggregate Functions
- Creating Views

Key Analysis:
- Total cases vs total deaths
- Infection percentage by population
- Countries with highest death counts
- Global COVID statistics
- Vaccination progress analysis

Tools Used:
SQL Server
GitHub
